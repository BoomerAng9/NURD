## Bug fixes shipped during Phase 1 LIVE deploy

- **PR #6** — `fix(infra): remove external traefik network — Traefik on myclaw-vps uses host mode`. Caught when `docker compose up` would have failed: the original compose declared `external: true` against a non-existent `traefik` network. Fixed by letting compose create `nurdscode_default` bridge.
- **PR #7** — `fix(auth): use Drizzle-managed 'sessions' table; stop connect-pg-simple collision`. Caught during the smoke test. Drizzle's `sessions` table and connect-pg-simple's auto-created `session` table both used index name `IDX_session_expire` → CREATE INDEX collision → every auth route 500. Fixed by pointing connect-pg-simple at the existing Drizzle table.
